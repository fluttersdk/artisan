# fluttersdk_artisan example

A minimal Flutter consumer app that wires `fluttersdk_artisan` as its CLI dispatcher.

This example was generated by `dart run fluttersdk_artisan install` and represents the canonical shape every Flutter app takes when it adopts artisan: a tiny `bin/dispatcher.dart` wrapper, a `lib/app/` codegen barrel surface, and an in-workspace plugin under `packages/awesome_plugin/`.

## What lives where

- `bin/artisan.dart` — the consumer wrapper. Imports `fluttersdk_artisan`, registers consumer commands from `lib/app/commands/_index.g.dart`, expands plugin providers from `lib/app/_plugins.g.dart`, then calls `runArtisan(args)`.
- `lib/app/commands/_index.g.dart` — codegen barrel. Regenerated by `make:command` and `commands:refresh`. Lists every consumer command class.
- `lib/app/_plugins.g.dart` — codegen barrel. Regenerated by `plugin:install <name>` and `plugins:refresh`. Lists every registered plugin provider.
- `packages/awesome_plugin/` — a workspace-resolved sample plugin scaffolded by `make:plugin awesome_plugin`. Demonstrates the `ArtisanServiceProvider` contract and the `install.yaml` schema.

## Try it

From this directory:

```bash
flutter pub get
dart run artisan list           # show every registered command
dart run artisan doctor         # validate environment
dart run artisan start          # run the Flutter app under artisan supervision
```

To exercise the MCP server (stdio JSON-RPC):

```bash
dart run fluttersdk_artisan:mcp
```

The full surface (10 substrate tools + per-plugin tools, idempotent installer DSL, scaffolders) is documented at [fluttersdk.com/artisan](https://fluttersdk.com/artisan).

## Note on the archive

This `example/` ships with the pub.dev package archive trimmed: only `lib/`, `bin/`, `packages/awesome_plugin/lib/`, `pubspec.yaml`, and this README. The full Flutter platform scaffold (`android/`, `ios/`, `macos/`, `linux/`, `windows/`, `web/`) is excluded by `.pubignore` to keep the archive under 500 KB. Clone the GitHub repository for a runnable end-to-end example.
